#### 一、so that 用法

`主` + `谓` + `so` + `adj.` + that + `从句`  
`主` + `谓` + `such` + `a/an` + `adj.` + `n. (单数)` + that + `从句`  
`主` + `谓` + `such` + `adj.` + `n. (复数)` + that + `从句`  
`主` + `谓` + `so` + `adj.` + `a/an` + `n. (单数)` + that + `从句`  
`主` + `谓` + `so` + `many/much/few/little` + `n.` + that + `从句`

variation `n.` `变化、变量`
